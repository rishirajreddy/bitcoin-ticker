# bitcoin_ticker

A Simple Crypto Currency Converter to convert Cryptocurrencies like BTC,ETH and LTC to Fiat currencies.

## Acknowledgment

Used <b><u>coinapi.io</u></b> for the api key which is only valid for 100 requests per day
